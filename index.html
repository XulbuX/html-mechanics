About this Project

Project demonstrates different HTML mechanics
and components with a clean, modern design.

The site is designed to be responsive
across different screen sizes.

Each page showcases a specific HTML element
with practical examples and styling.


This is website does not make any use of
frameworks like Bootstrap or TailwindCSS.
It is built entirely with semantic
HTML5 and modern CSS techniques.
